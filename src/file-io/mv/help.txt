Usage: mvf [OPTION]... SOURCE DEST
  or:  mvf [OPTION]... SOURCE... DIRECTORY
  or:  mvf [OPTION]... -r DIRECTORY DIRECTORY
Rename SOURCE to DEST, or move SOURCE(s) to DIRECTORY.

Options:
    -r, --recursive     move directories recursively

    -i, --interactive   prompt if permission is needed or DEST already exists

    -f, --force         lorem ipsum

    -v, --verbose       print operations being done

    -h, --help          display this help message and exit

    --version           display CU-Fied version and exit
